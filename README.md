# Kode

Kode is an AI IDE based on Code - OSS, focused on helping Yudao users understand, start, debug, and extend their projects faster.

This repository is the Kode product codebase. It keeps the Code - OSS foundation while adding Kode-specific branding, themes, and future AI workflows for Yudao / ruoyi-vue-pro / yudao-cloud development.

## Goals

- Make Yudao projects easier to start locally.
- Help users diagnose startup and runtime errors from logs.
- Generate module development plans that match Yudao conventions.
- Explore focused AI IDE workflows that can inform broader coding-agent products.
